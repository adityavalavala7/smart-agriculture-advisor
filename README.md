# 🌾 Smart Agriculture Advisor

## Overview

Smart Agriculture Advisor is an AI-powered farming assistant developed using Python, Gradio, and Gemini AI. The application helps farmers and agriculture enthusiasts make informed decisions by providing crop recommendations, agriculture-related guidance, weather-based farming advice, and disease information with remedies.

## Features

### 🌱 Crop Recommendation

* Recommends suitable crops based on location, soil type, and season.
* Provides brief reasons for each recommendation.

### 🤖 Agriculture Chatbot

* Answers agriculture-related questions.
* Provides practical farming guidance and recommendations.

### 🌦 Weather-Based Farming Advice

* Fetches real-time weather information.
* Suggests irrigation, fertilizer, and pest-control practices based on weather conditions.

### 🦠 Disease Information and Remedies

* Provides information about crop diseases.
* Explains symptoms, causes, remedies, and prevention methods.

## Technologies Used

* Python
* Gradio
* Google Gemini AI
* Requests Library
* Weather API (wttr.in)

## Installation

1. Clone the repository:

```bash
git clone <repository-url>
```

2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Open `app.py` and replace:

```python
YOUR_GEMINI_API_KEY
```

with your own Gemini API key.

4. Run the application:

```bash
python app.py
```
